About

EvoDevo Papers was inspired by the pioneering @fly_papers (RIP) and @phy_papers. I created the initial version based on the fly_papers post and the phy_papers documentation, using automation services like dlvrit, IFTTT, and Zapier. It was active as a Twitter bot from 2015â€“2023. After Twitter's takeover and API changes, the bot stopped working.

Since 2022, EvoDevo Papers is running on Mastodon. The first version used feed2toot, but due to problems with this setup, I re-built the bot from scratch to improve robustness and flexibility. From January 2024, it is powered by:

  • Django (a Python web framework)
  • django-feed-reader (a Django package for feed consumption)
  • django-distill (a Django package for generating static websites)
  • Mastodon.py (a Python library for the Mastodon API)
  • Simple.css (a CSS framework for semantic HTML)
