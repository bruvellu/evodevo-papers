Acknowledgments

EvoDevo Papers was inspired by the pioneering @fly_papers (RIP) and @phy_papers. I created the initial version based on the phypapers documentation using automation services like dlvrit, IFTTT, and Zapier. It was active as a Twitter bot from 2015â€“2023, until changes in the Twitter API limited its functionalities. The Mastodon bot version exists since December 2022. I ran it using feed2toot until December 2023, when it stopped working. This motivated me to re-build the bot from scratch in January 2024 to improve robustness and flexibility. The current version of EvoDevo Papers is powered by:

  • Django (a Python web framework)
  • django-feed-reader (a Django package for feed consumption)
  • django-distill (a Django package for generating static websites)
  • Mastodon.py (a Python library for the Mastodon API)
  • Simple.css (a CSS framework for semantic HTML)
