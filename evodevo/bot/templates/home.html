About

EvoDevo Papers was inspired by the pioneering @fly_papers (RIP) and @phy_papers. I created the initial version based on the fly_papers post and the phy_papers documentation, using automation services like dlvrit, IFTTT, and Zapier. It operated as a Twitter bot from 2015 to 2023, but after the takeover and subsequent API changes, it could no longer post there.

Meanwhile, EvoDevo Papers has been posting on Mastodon since 2022. The first version was based on feed2toot. However, due to recurrent issues with this setup, I rebuilt the bot from scratch to enhance its robustness and flexibility. The new version, which has been running smoothly since January 2024, is powered by:

  • Django (a Python web framework)
  • django-feed-reader (a Django package for feed consumption)
  • django-distill (a Django package for generating static websites)
  • Mastodon.py (a Python library for the Mastodon API)
  • Simple.css (a CSS framework for semantic HTML)
